# AI-Agent-Business-Process-Automation

## Project Overview
This project implements an AI agent that automates a business monitoring and reporting process.

## Agent Capabilities
- Multi-step reasoning using ReAct pattern
- External information retrieval
- Data processing and analysis
- Human approval workflow

## Tools Used
1. Web Search Tool
2. Python Analysis Tool
3. Snapshot/Diff Tool

## Files
- AI_Agent_Prototype.py: Working prototype
- Framework_Justification.md: Framework comparison
- Agent_Execution_Traces_and_Evaluation.md: ReAct traces and evaluation
- Design_and_Evaluation_Report.md: Final report
